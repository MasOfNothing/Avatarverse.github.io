Cooking

Avatarverse has a custom cooking system that allows you to slowly cook food in a special cooking pot. This system allows for more complex recipes and is similar to the brewing system. To cook food, you need to craft a cooking pot, which is a campfire under 5 iron ingots, almost like the crafting recipe for an iron helmet. Once you have a cooking pot, you can place it down. To cook food, you need to right-click on it with the food you want to cook. A hologram will appear above the cooking pot showing the progress bar. When the food is done cooking, it will appear above the cooking pot. You can then right-click on the food to pick it up (and get XP for it).

When the ingredients you cook don't match an existing recipe, you instead make a stew called "Street Gruel". Instead of listing all the recipes below, we recommend you visit a library in game to find Master Chef Mushi's famous cookbook. This cookbook contains all the recipes you can make in the game.
